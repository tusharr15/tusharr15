### 🌿 Farm2Formula – Blockchain-Based Herb Tracking App

Developed a blockchain-powered mobile application to track Ayurvedic herbs from farm to final product, ensuring supply chain transparency. Implemented GPS geo-tagging for each herb batch recorded on blockchain, making data tamper-proof and verifiable via QR code scanning. Collaborated on database schema design with normalization to handle multi-stakeholder data across farmers, labs, and manufacturers.

| Layer      | Technology  |
|------------|-------------|
| Frontend   | React (mobile) |
| Data Layer | Blockchain, QR-based verification |
| Database   | Normalized multi-stakeholder schema |
